# superdonut CLI

A CLI for Donut's Template.

[Learn More](https://github.com/heybrunoandrade/react-donuts-template)

## Install

```bash
 npm i -g superdonut
```

```bash
// Configure Eslint
 superdonut configure:eslint

// Create Component
 superdonut generate:component MyComponent

// Create Page
 superdonut generate:page Main

```
