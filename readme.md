# superdonut CLI

A CLI for Donut's Template.

[Learn More](https://github.com/heybrunoandrade/react-donuts-template)

## Install

```bash
 npm i -g superdonut
```

```bash
// Configure Eslint
 superdonuts configure:eslint

// Create Component
 superdonuts generate:component MyComponent

// Create Page
 superdonuts generate:page Main

```
